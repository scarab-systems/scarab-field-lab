# Assets

This folder contains public-facing Scarab Field Lab visual assets.

## Mascot

- `scarab-mascot.png`: approved Scarab Systems mascot image for subtle
  Field Lab trust-mark use.

Use the mascot only when the target context welcomes a friendly image. It is
appropriate for repos or public comments that explicitly invite a light touch.
It should not be used for security-sensitive reports, contentious maintainer
threads, or as a substitute for evidence and validation.
