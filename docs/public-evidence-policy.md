# Public Evidence Policy

Public case files should contain enough evidence for trust while staying limited
to public records.

## Public-Safe Evidence

- Issue and PR URLs.
- Public repository names.
- Public file paths changed in an upstream patch.
- Validation summaries.
- Sanitized diagnostic summaries.
- Public case status.

## Excluded Evidence

- Non-public records.
- Non-public SDS materials.
- Target repository clones.
- Secrets or private correspondence.

## Review Rule

Before publication, review every case page for public evidence, claim limits,
and private-material leakage.
