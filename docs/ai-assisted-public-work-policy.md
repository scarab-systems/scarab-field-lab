# AI-Assisted Public Work Policy

AI assistance is allowed in Scarab Systems public field-test work only after
diagnostic evidence exists. This policy does not open Scarab Field Lab to
unsolicited issues or pull requests.

## Allowed

- Drafting a narrow patch from documented evidence.
- Summarizing public diagnostic findings.
- Organizing validation notes.
- Drafting public explanations for human review.

## Not Allowed

- Using AI to invent diagnostic truth.
- Publishing unattended generated patches.
- Treating an AI response as validation.
- Replacing maintainer review with agent confidence.
- Publishing non-public SDS materials.

## Disclosure

Each case file includes an assistance disclosure. The disclosure does not
need to name every tool used. It must make clear that SDS findings are
mechanical and that any assisted implementation work remains human reviewed.
